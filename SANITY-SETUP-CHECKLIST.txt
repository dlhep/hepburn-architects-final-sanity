HEPBURN ARCHITECTS — SANITY CMS SETUP

1. Create/sign in to Sanity and create a project.
2. Use dataset name: production (public dataset).
3. Copy the Project ID.
4. Sanity Manage > API > CORS Origins:
   Add https://www.hepburnarchitects.net and allow credentials.
5. Vercel > Project > Settings > Environment Variables:
   NEXT_PUBLIC_SANITY_PROJECT_ID = your project ID
   NEXT_PUBLIC_SANITY_DATASET = production
   SANITY_REVALIDATE_SECRET = a long random secret
6. Redeploy Vercel.
7. Open https://www.hepburnarchitects.net/studio and sign in.
8. Add/edit projects and click Publish.

The project records and uploaded images are stored by Sanity, not GitHub.
Future website ZIP uploads will not overwrite Sanity content.

The old /admin address redirects to /studio.
See README.md for importing the existing five projects and adding the optional instant-update webhook.
