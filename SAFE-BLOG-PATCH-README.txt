SAFE SANITY BLOG PATCH

This patch only adds the Sanity article schema, blog routes, and a merged guide system.
It does not replace the homepage, layout, header, footer, global CSS, sitemap, or existing guide data.

After deployment:
- /studio will show “Guides & Blog Posts”
- Sanity blog posts appear at /blog
- New Sanity guides are added to /guides
- Existing static guides remain in place
- Reusing an existing guide slug will not create a duplicate listing

Content normally appears within about 60 seconds after publishing.
